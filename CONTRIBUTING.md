# Contributing

Thanks for your interest in contributing to edgeprobe.

## Development

1. Fork the repo and create your branch from `main`.
2. Make your changes.
3. Run tests:

```sh
go test ./...
```

4. Open a pull request with a clear description of the problem and solution.

## Guidelines

- Keep changes focused and small.
- Add tests for new behavior or fixes.
- Keep the existing style and conventions.

## Reporting Bugs / Requesting Features

Please use GitHub Issues and include:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Environment details (OS, Go version)
